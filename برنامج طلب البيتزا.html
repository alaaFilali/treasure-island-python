برنامج طلب البيتزا

التعليمات:

اكتب برنامج بقوم بأتمتة عملية بيع البيتزا.

بناءا على طلب المستخدم احسب الفاتورة النهائية له:

  • البيتزا الصغيرة 15 دولار

  • البيتزا المتوسطة 20 دولار

  • البيتزا الكبيرة 25 دولار

  • اضافة pepperoni للبيتزا الصغيرة زيادة 2 دولار

  • اضافة pepperoni للبيتزا المتوسطة والكبيرة زيادة 3 دولار

  • زيادة الجبنة لاي نوع بيتزا زيادة 1 دولار

انواع البيتزا تمثل بالاحرف S للصغيرة و M للمتوسطة و L للكبيرة وان كان يريد اضافة pepperoni او الجبنة سوف يدخل اما Y لنعم او N للا

والنتيجة النهائية للبرنامج سوف تكون كالتالي: 


Welcome to Python Pizza Deliveries!
What size pizza do you want? S, M or L: M
Do you want pepperoni? Y or N: Y 
Do you want extra cheese? Y or N: Y
Your final bill is $24

اليك هذه الاسطر كبداية وحظا موفقا في حل التمرين. 


print("Welcome to Python Pizza Deliveries!")
size = input("What size pizza do you want? S, M or L: ")
add_pepperoni = input("Do you want pepperoni? Y or N: ")
extra_cheese = input("Do you want extra cheese? Y or N: ")

الحل:

اولا سوف نتحقق من نوع البيتزا التي اختارها وسوف اتتبع الفاتورة في متغير اعرفه من قبل 


bill = 0

if size == "S":
    bill = 15
elif size == "M":
    bill = 20
else:
    bill = 25

والان سوف نتحقق ان كان يريد اضافة peperroni اول لا فان كانت الاجابة نعم سوف نتحقق من نوع البيتزا لان السعر مختلف 


if add_pepperoni == "Y":
    if size == "S":
        bill += 2
    else:
        bill += 3

واخيرا سوف نتحقق ان كان يريد اضافة جبنة اضافية 


if extra_cheese == "Y":
    bill += 1

وفي الاخير اقوم بطباعة الفاتورة النهائية: 


print("Welcome to Python Pizza Deliveries!")
size = input("What size pizza do you want? S, M or L: ")
add_pepperoni = input("Do you want pepperoni? Y or N: ")
extra_cheese = input("Do you want extra cheese? Y or N: ")

bill = 0

if size == "S":
    bill = 15
elif size == "M":
    bill = 20
else:
    bill = 25

if add_pepperoni == "Y":
    if size == "S":
        bill += 2
    else:
        bill += 3

if extra_cheese == "Y":
    bill += 1

print(f"Your final bill is ${bill}")
